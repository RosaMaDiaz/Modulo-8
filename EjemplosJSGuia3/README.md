## 📖 Contenido de la Guía

### Unidad 1: Arreglos en JavaScript

- Concepto y utilidad de los arreglos.
- Creación de arreglos: vacíos, con datos iniciales, numéricos y mixtos.
- Acceso y modificación de elementos mediante índices.
- Propiedad `length` y su importancia.
- Recorridos con `for`, `while` y `for...of`.
- Métodos básicos: `push()`, `pop()`, `shift()`, `unshift()`, `splice()`.
- Arreglos multidimensionales y aplicaciones prácticas (tablas, calificaciones, inventarios).
- Buenas prácticas: nombres descriptivos, validación de posiciones, ciclos adecuados.

### Unidad 2: Objetos en JavaScript

- Relación con la Programación Orientada a Objetos.
- Creación de objetos con `new Object()` y literales.
- Propiedades, métodos y manipulación de objetos.
- Objetos del lenguaje, del navegador (BOM) y del documento (DOM).
- JSON vs Objetos en JavaScript.
- Ejercicios integradores: gestión de vehículos, cálculo de edad con `Date`.

### Unidad 3: Aplicaciones Dinámicas con Objetos y DOM

- Introducción a aplicaciones dinámicas.
- Manipulación del DOM: `getElementById()`, `querySelector()`, `innerHTML`, `textContent`.
- Creación dinámica de elementos con `createElement()` y `appendChild()`.
- Uso de `classList` para gestionar clases CSS.
- Eliminación de elementos con `remove()`.
- Integración de objetos y arreglos con el DOM.
- Generación dinámica de listas, tarjetas y tablas.
- Manejo de eventos (`onclick`, `oninput`) para actualización en tiempo real.
- Actividades integradoras: registro dinámico de participantes, sistema de cursos.
