# 📘 Guía de Laboratorio #1 – Módulo 8: Fundamentos de JavaScript

## 📌 Información General

- **Nombre de la práctica:** Fundamentos de JavaScript y estructuras de control
- **Modalidad:** Virtual
- **Tiempo estimado:** 4 horas
- **Materia:** Programación con JavaScript
- **Docente:** Lic. Ramón Morillo Ullola
- **Estudiante:** Rosa Ma. Díaz

---

## 🎯 Resultado de Aprendizaje

El participante utiliza **tipos de variables, operadores y estructuras de control de flujo** para estructurar datos en JavaScript.

---

## 🏆 Competencia Final del Módulo

Al finalizar el módulo, el estudiante será capaz de desarrollar **aplicaciones básicas e interactivas del lado del cliente** para páginas web, manipulando contenido dinámico, controlando eventos, validando formularios y gestionando elementos del DOM, aplicando buenas prácticas de programación.

---

## 📖 Desglose Didáctico

El participante:

- Identifica fundamentos y sintaxis básica de JavaScript.
- Utiliza variables, operadores y estructuras de control.
- Implementa funciones y estructuras repetitivas.
- Manipula elementos del DOM de forma dinámica.
- Gestiona eventos e interacciones en páginas web.
- Utiliza la variable `this` y propiedades básicas de JavaScript.
- Desarrolla validaciones de formularios.
- Integra JavaScript en páginas web aplicando buenas prácticas.
