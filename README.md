# 📘 Módulo 8: Programación con JavaScript

Este repositorio resume las **Guías de Laboratorio #1, #2, #3 y #4** del Módulo 8, orientadas al aprendizaje de JavaScript para el desarrollo de aplicaciones web dinámicas e interactivas.

---

## 🎯 Competencia Final del Módulo
Al finalizar, el participante será capaz de:
- Desarrollar aplicaciones básicas del lado del cliente.
- Manipular contenido dinámico con JavaScript.
- Controlar eventos y validar formularios.
- Gestionar elementos del DOM aplicando buenas prácticas.

---

## 📖 Guía 1: Fundamentos de JavaScript
- Sintaxis básica, variables y operadores.
- Estructuras de control (`if`, `else`, `switch`).
- Funciones y alcance de variables.
- Uso de `this` y contexto de ejecución.
- Introducción a ECMAScript y TypeScript.
- Scripts internos y externos en HTML.
- Ejemplos iniciales: `alert()`, `prompt()`, `confirm()`.

---

## 📖 Guía 2: (Bloqueada)
- Contenido no disponible por restricciones.
- Se asume continuidad entre fundamentos y estructuras más avanzadas.

---

## 📖 Guía 3: Aplicaciones Dinámicas con DOM
- **Arreglos**: creación, recorrido (`for`, `while`, `for...of`), métodos (`push`, `pop`, `splice`).
- **Objetos**: propiedades, métodos, JSON, integración con DOM.
- **Manipulación del DOM**:
  - `getElementById()`, `querySelector()`.
  - `innerHTML`, `textContent`.
  - Creación dinámica con `createElement()` y `appendChild()`.
- **Aplicaciones prácticas**:
  - Listas y tarjetas dinámicas.
  - Tablas generadas desde arreglos/objetos.
  - Eventos (`onclick`, `oninput`) para actualización en tiempo real.

---

## 📖 Guía 4: Formularios, Tablas y CRUD Básico
- **Formularios**:
  - Captura de datos con `value`, `checked`, `selectedIndex`.
  - Validaciones: campos vacíos, longitud mínima, numéricos, correos.
  - Uso de `preventDefault()` para controlar envíos.
- **Tablas dinámicas**:
  - Generación de filas con `createElement()` y `appendChild()`.
  - Mostrar información desde arreglos de objetos.
  - Actualización y limpieza de contenido.
- **CRUD básico**:
  - Crear, Leer, Actualizar y Eliminar registros.
  - Confirmación de acciones y mensajes dinámicos.
- **Ejercicios integradores**:
  - Registro de productos/libros/vehículos.
  - Gestión de empleados y pedidos.

---

## 🛠️ Buenas Prácticas
- Separar HTML, CSS y JavaScript.
- Usar nombres descriptivos en variables y funciones.
- Comentar procesos importantes.
- Validar datos antes de procesarlos.
- Mantener código modular y reutilizable.
