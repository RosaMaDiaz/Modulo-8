**Gestión de Datos con JavaScript: Formularios, Tablas Dinámicas y CRUD Básico**

---

## 📌 Información General

- **Modalidad:** Virtual
- **Tiempo estimado:** 8 horas

## 🎯 Resultado de Aprendizaje

Desarrollar aplicaciones web dinámicas capaces de:

- Capturar información mediante formularios.
- Procesar y validar datos ingresados por el usuario.
- Almacenar información en arreglos de objetos.
- Presentar datos en tablas dinámicas generadas con JavaScript.
- Implementar operaciones CRUD (Crear, Leer, Actualizar y Eliminar).

---

## 🏆 Competencia Final del Módulo

Al finalizar el módulo, el participante será capaz de:

- Desarrollar aplicaciones básicas e interactivas del lado del cliente.
- Manipular contenido dinámico con JavaScript.
- Validar formularios y gestionar elementos del DOM.
- Aplicar buenas prácticas de programación web.

---

## 📂 Contenido de la Guía

1. **Formularios y Captura de Datos**
   - Obtención de valores con `value`, `checked`, `selectedIndex`.
   - Validación de campos vacíos, longitud mínima, numérica y correos electrónicos.
   - Uso de `preventDefault()` para controlar envíos.
   - Ejercicio integrador: Formularios y eventos en JavaScript.

2. **Tablas Dinámicas**
   - Creación de filas con `createElement()` y `appendChild()`.
   - Mostrar información desde arreglos de objetos.
   - Actualización y limpieza de tablas.
   - Ejercicio integrador: Sistema de registro (productos, libros, vehículos).

3. **Operaciones CRUD Básicas**
   - Crear, Leer, Actualizar y Eliminar registros.
   - Confirmación de acciones y mensajes informativos.
   - Ejercicios integradores: Gestión de empleados, pedidos de pizza, anuncios dinámicos.
